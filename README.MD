# Just Dance (Rhythm Game)
Fundamental Game Design
It's a music game developed by Java. Features include:
1. Music selection
2. XBox Gamepad controller support
3. Keyboard controller support
4. Scoring system
5. Special effects
6. BPM detection
7. Audio synchronization 
8. more to come
